# CH32X035 ピン情報・電気的仕様 (MimicX 用ノート)

WCH CH32X035 データシート V1.7 (`CH32X035.pdf`) からの抜粋。
MimicX シリーズの全基板で共通 MCU として使用するため、各バリエーションの
回路設計時にこのファイルを参照する。

PDF 出典:
- `CH32X035.pdf` (データシート V1.7、本文 42 頁)
- `CH32X035RM.PDF` (リファレンスマニュアル — 周辺機能レジスタ詳細)

|---|---|---|---|---|---|---|---|---|---|
| CH32X035R8T6 | LQFP64M | 10×10 mm | 0.5 mm | 60 | 14+1 | 2 | 3 | ✓ | ✓ |
| CH32X035C8T6 | LQFP48 | 7×7 mm | 0.5 mm | 46 | 10+1 | 2 | 3 | ✓ | ✓ |
| **CH32X035G8U6** | **QFN28** | **4×4 mm** | **0.4 mm** | **27** | **10+1** | **2** | **1** | **✓** | **✓** |
| CH32X035G8R6 | QSOP28 | 3.9×9.9 mm | 0.635 mm | 26 | 11+1 | 2 | 3 | ✓ | ✓ |
| CH32X035F8U6 | QFN20 | 3×3 mm | 0.4 mm | 19 | 10+1 | 2 | – | – | ✓ |
| CH32X035F7P6 | TSSOP20 | 4.4×6.5 mm | 0.65 mm | 18 | 11+1 | 1 | 1 | – | ✓ |

共通仕様: Flash 62 KB / SRAM 20 KB、TIM 高度2 + 汎用1、USART ×4、I2C、SPI、
USB 2.0 FS PHY 内蔵、PIOC、Watchdog ×2。

> ⚠️ MimicX-hardware の README には旧版で「F8U6 = TSSOP-20」と書かれて
> いるが、データシート上 **F8U6 は QFN20** (3×3 mm) が正。20 ピン TSSOP は
> F7P6 で型番が異なる。

## 2. 型番の読み方 (Series Product Naming Rules)

```
CH32  X  03  5  G   8   U   6
↑     ↑  ↑   ↑ ↑   ↑   ↑   ↑
|     |  |   | |   |   |   └ 温度範囲 (6: -40〜85℃)
|     |  |   | |   |   └─── パッケージ (T:LQFP / U:QFN / R:QSOP / P:TSSOP / M:SOP)
|     |  |   | |   └─────── Flash サイズ (8 = 64KB)
|     |  |   | └─────────── ピン数 (J:8 / A:16 / F:20 / G:28 / K:32 / T:36 / C:48 / R:64 / W:68 / V:100 / Z:144)
|     |  |   └───────────── サブファミリ (35: USB + USB PD/Type-C 接続性)
|     |  └───────────────── 製品タイプ (0 = QingKe V2/V4、≤48MHz)
|     └──────────────────── デバイスファミリ (X = RISC-V 専用 IO)
└────────────────────────── ベース (CH32 = WCH 32-bit MCU)
```

### 内部短絡されているピン (重要)

データシート Note 4 / Note 5:

- **Pin 16: PB1 と PB5 が内部で短絡 & 封止** (G8U6 / G8R6 共通)。両方を
  出力モードに設定することは禁止。
- **Pin 26: PC16 と PC11 が内部短絡**。Pin 27: **PC17 と PC10 が内部短絡**。
  両方を出力モードに設定することは禁止。USB アプリケーションでは PC10 /
  PC11 をフローティング入力 (リセット直後の既定値) に維持すること。
- **Pin 4 (PC3) は QSOP28 / TSSOP20 でのみ RST 機能あり**。QFN28 (G8U6)
  では RST 不可、通常 GPIO として扱う。

### atari-joystick 固有割当 (ATARI / X68000 標準)

ホスト機側 D-SUB 9pin はオス、当基板側はメス (CONNFLY DS1037-09FNAKT74-0CC)。

| ATARI 信号 | D-SUB 9pin | QFN28 Pin | MCU 信号 | 備考 |
|---|---:|---:|---|---|
| Up | 1 | 6 | PA1 | open-drain (FW 設定) |
| Down | 2 | 7 | PA2 | 同上 |
| Left | 3 | 8 | PA3 | 同上 |
| Right | 4 | 9 | PA4 | 同上 |
| +5V (host) | 5 | — | — | 通常 NC |
| TRIG-A | 6 | 11 | PA6 | open-drain (FW 設定) |
| TRIG-B | 7 | 12 | PA7 | 同上 |

## 5. 電気的仕様 (抜粋)

### 絶対最大定格 (Table 3-1)

| 項目 | 最小 | 最大 | 単位 |
|---|---:|---:|---|
| 動作周囲温度 TA | -40 | 85 | ℃ |
| 保管温度 TS | -40 | 125 | ℃ |
| VDD 電圧 | -0.3 | 6.0 | V |
| 各 IO ピン VIN | -0.3 | VDD+0.3 | V |
| ESD (HBM) | — | 4 | kV |
| VDD ピン総電流 IVDD | — | 150 | mA |
| GND ピン総電流 IGND | — | 200 | mA |
| 各 IO シンク電流 | — | 40 | mA |
| 各 IO ソース電流 | — | 30 | mA |

### 推奨動作条件 (Table 3-2)

| 項目 | 条件 | 最小 | 最大 | 単位 |
|---|---|---:|---:|---|
| FSYS (内部バス周波数) | — | — | 48 | MHz |
| VDD | USB / PD 無効 | 2.0 | 5.5 | V |
| VDD | **USB / PD 有効** | **3.0** | **5.3** | **V** |
| VDD | ADC 有効 | 2.5 | 5.5 | V |

> MimicX は USB バスパワー (5V) で動作させるので USB 有効時の VDD 範囲
> (3.0〜5.3 V) を満たす。5 V トレラント GPIO のため、ATARI 側ホストの 5V
> 信号は直結可。
